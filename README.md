# 概要

本システムは、動物園や研究機関における観察業務を支援するために開発した **「動物行動ログ分析・可視化システム」** です。

観察画像をアップロードすると、AIが動物の行動を分類し、その結果を行動ログとして保存します。

さらに、

* 個体別管理
* ダッシュボードによる可視化
* 異常傾向分析
* AIによる日本語観察レポート生成

までを一つの業務フローとして実現しています。

本プロジェクトは単なるAI分類デモではなく、

* AI
* Dashboard
* Data Analysis
* Business System Design

を組み合わせた業務システムを意識して設計・開発しました。

# デモ環境について

公開デモ環境（Render）では、無料プランのメモリ容量および実行時間の制約を考慮し、AI推論処理を軽量化した **Demo Mode** で動作しています。

一方、ローカル環境では TensorFlow モデルを利用した画像分類機能を実装しており、画像解析から行動ログ生成、ダッシュボード表示、異常傾向分析、AI観察レポート生成までの一連の処理を実行できます。

本プロジェクトの目的は、AIモデル単体の分類精度を示すことではなく、

* AI画像分類
* 行動ログ管理
* Dashboardによる可視化
* OpenAI API連携
* 業務システム設計

を統合した **動物行動ログ分析・可視化システム** を構築することにあります。

そのため、公開デモではシステム全体の業務フローを体験できる構成とし、ローカル環境ではTensorFlowモデルによる実際の画像分類機能を実装しています。
