# Animal Behavior Analytics System

## 動物行動ログ分析・可視化システム

---

# 概要

本システムは、動物園や研究機関における観察業務を支援するために開発した「動物行動ログ分析・可視化システム」です。

動画をアップロードするだけで、

* 行動分類
* 行動ログ生成
* ダッシュボード表示
* 異常分析
* AI観察レポート生成

までを自動で実行します。

単なるAIデモではなく、実際の業務システムを意識して設計・開発しました。

---

# システム構成

```text
動画アップロード

↓

フレーム抽出（OpenCV）

↓

行動分類（TensorFlow）

↓

行動ログ生成（CSV）

↓

ダッシュボード表示

↓

異常分析

↓

AI観察レポート生成（OpenAI API）
```

---

# 主な機能

## 動画解析

* 動画アップロード
* 観察セッション作成
* フレーム抽出
* 自動解析

---

## AI行動分類

TensorFlowモデルを利用して動物の行動を分類します。

分類内容

* sleeping（睡眠）
* eating（食事）
* land_activity（陸上活動）
* water_activity（水中活動）

---

## 観察セッション

管理項目

* Animal ID
* 観察開始時刻
* フレーム取得間隔
* 動画ファイル
* 行動ログ

---

## 行動ログ

CSV形式で保存します。

保存項目

* datetime
* animal_id
* start_time
* frame_number
* behavior
* confidence

---

# ダッシュボード

## 行動サマリー

表示内容

* 睡眠
* 食事
* 陸上活動
* 水中活動

---

## 行動割合

Pie Chart により行動割合を可視化します。

---

## 行動タイムライン

例

```text
09:00:00　睡眠

09:00:10　睡眠

09:00:20　食事

09:00:30　陸上活動
```

---

## 個体別行動サマリー

例

```text
Capybara-01

睡眠　　　3件

食事　　　1件

陸上活動　2件

水中活動　0件
```

---

# 異常分析

現在はルールベースによる分析を実装しています。

例

```text
睡眠割合 ＞ 60％

↓

睡眠割合が高い可能性があります。
```

今後は統計分析やAIによる異常検知への拡張を予定しています。

---

# AI観察レポート

行動サマリーをもとに、

OpenAI API を利用して日本語の観察レポートを自動生成します。

処理の流れ

```text
行動サマリー

↓

プロンプト生成

↓

OpenAI API

↓

日本語観察レポート生成
```

# 制作背景

動物園や研究施設では、動物の行動観察を人手で記録する場面が多く、観察コストや記録漏れが課題となっています。

そこで本システムでは、

* 動画解析
* 行動分類
* 行動ログ管理
* データ可視化
* AIレポート生成

を一つの業務フローとして統合し、観察業務の効率化を目的として開発しました。
